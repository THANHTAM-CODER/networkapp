HOST

HOST: Trong lĩnh vực mạng máy tính, host (hay network host) là bất kỳ thiết bị nào (máy tính, điện thoại, máy chủ) được kết nối vào mạng và sở hữu một địa chỉ IP riêng. Host đóng vai trò là nguồn gửi, đích nhận hoặc trạm trung chuyển dữ liệu trong hệ thống mạng.


➜ Nói tóm lại HOST là thiết bị được kết nối vào mạng và sở hữu IP riêng


🌐 Các loại Host phổ biến

  • Client Host: Các thiết bị người dùng cuối như máy tính, laptop, điện thoại, hoặc máy tính bảng dùng để truy cập mạng.
  • Server Host (Máy chủ): Các máy tính có cấu hình mạnh, cung cấp tài nguyên, dịch vụ hoặc lưu trữ dữ liệu để các thiết bị khác truy cập.
  • Thiết bị ngoại vi thông minh: Máy in mạng, camera IP hoặc các thiết bị IoT cũng được coi là một host nếu chúng giao tiếp được trên cùng hệ thống.

🌐 Phân biệt Host, Server và Hosting

  • Host (Network Host): Chỉ định một thiết bị tham gia và giao tiếp trong mạng.
  • Server: Là thiết bị phần cứng hoặc phần mềm chuyên dụng chuyên cung cấp dịch vụ.
  • Hosting (Web Hosting): Là dịch vụ thuê không gian lưu trữ trên máy chủ để vận hành website hoặc ứng dụng.
